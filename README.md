Our website UNISPHERE provides multiple features including academic progress tracking,
wellness support, career pathway guidance, peer mentoring, study groups, and an
interactive campus map for navigation.

Features
User Authentication
 Login with email and password
 Account registration and password recovery
Dashboard
 Current CGPA/marks display
 Hydration tracker with motivational tips and reminders
 Study timer using Pomodoro technique with weekly progress
 Wellness streak tracker for building consistent study habits
 Academic progress trend charts displaying marks, assignments, and tests weekly
Academic Hub
 Learn Mode: Free lessons with interactive rich media content and self-assessment
quizzes (e.g., Data Structures Fundamentals)
 Quiz Mode: Timed quizzes with instant scoring; free for all users
 Race Mode (Premium): Real-time multiplayer quiz races with leaderboards, badges,
and performance analytics
 Match Mode (Premium): Team-based competitive problem-solving with collaborative
tools and voice/video chat
Active Courses
 List and track progress in enrolled courses (e.g., Mathematics 201, Physics 301, CS
401)
 Bar graphs for course performance by subject
Wellness Center
 Wellness check-in with mood tracking and AI chatbot for emotional assistance
 Mental health resources including counseling services, stress management tips,
emergency helplines, and wellness articles
 Appointment booking for counseling sessions with date/time and reason input
 Wellness activities tailored to student timetables (meditation, yoga, support groups)
 Mood trend graphs and wellness streak stats
Career Pathways
 Career events and virtual fairs listing

 Internship board with government/private opportunities, filtering by category
 Skill-building roadmaps and interview preparation guides
 Scholarship search and application tracking
 Career readiness scores visualized by skill areas (technical, communication,
leadership, teamwork, problem-solving)
Appointments
 Faculty, academic advisor, career counseling, and wellness counseling appointments
booking
 Scheduling interface with date, time, and topic selection
Financial Aid
 Emergency fund application and tracking
 Financial literacy content with budgeting and saving tools
 Education loans comparison and planning tools
 Scholarship finder with AI recommendation and deadline alerts
 Budget management with expense tracking and AI-driven insights
 Work-study program listings and job recommendations
Study Groups
 Browse and join active study groups by subject
 Create new study groups dynamically based on enrolled subjects
Peer Mentors
 List peer mentors ranked by CGPA and specialization
 Chat and book sessions with peer mentors
 Mentor rating system with response times and rates
E-Library &amp; Resources
 Access digital books, research papers, video lectures, study guides, online courses,
and virtual lab simulations
Community
 Post sharing and discussion forum
 Networking events such as tech meetups, alumni mixers, and career fairs
Campus Maps &amp; Navigation
 Interactive campus map with Maps API integration
 Location and directions to campus buildings, dining halls, recreation areas, and
parking lots

Chatbot Features
 Supports Create, Read, Update, Delete (CRUD) operations on suggested query
chips like:
 What&#39;s my next class?
 My scholarships
 Career queries
 Tutoring requests
 Campus navigation help
 Book appointments
 Motivational talks
 Emotional assistance
 Jokes and riddles related to subjects
 Chatbot retains key user information to avoid repetitive explanations
 Backend implementation (recommend Python or PHP) with necessary API routes
and error handling
